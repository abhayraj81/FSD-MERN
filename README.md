# FSD-MERN

A collection of practice work, assignments, and exercises completed while learning the fundamentals of full-stack web development, building toward the MERN stack (MongoDB, Express, React, Node.js).

This repository documents the early stages of that journey — starting with the building blocks of the web: structure, styling, and interactivity.

## Getting Started

Clone the repository and open any HTML file directly in a browser — no build step required.

```bash
git clone https://github.com/abhayraj81/FSD-MERN.git
cd FSD-MERN
```

Navigate into any folder and open the relevant `.html` file to view it locally.
